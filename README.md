# Moto Élite Care — Project Dashboard

Dashboard di project management per il lancio del primo centro Moto Élite Care a Milano.

## Deploy su Vercel

1. Importa questo repository su Vercel
2. Il sito sarà disponibile all'URL assegnato
3. I dati si salvano nel localStorage del browser

## Struttura
- `public/index.html` — La dashboard completa
- `vercel.json` — Configurazione deploy
